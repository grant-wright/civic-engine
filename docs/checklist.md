# Build Checklist

## Build Preferences

- **Build mode:** Step-by-step
- **Comprehension checks:** Yes — after each item, one question. Notes flagged for post-delivery follow-up.
- **Git:** Commit after each item. Message format: "Complete step N: [title]". Build agent walks through all git commands — Grant is new to Git.
- **Verification:** Yes — per-item. Run the app, confirm specific expected output before moving on.
- **Check-in cadence:** Learning-driven — discuss what was built, why it works that way, what to notice in the code.
